MedicaApp (desktop)
==================

Uruchom:
  python main.py

Pliki w folderze:
- emar.db: baza danych (tworzy się automatycznie obok main.py)
- Rejestr_Produktow_Leczniczych_*.csv: (opcjonalnie) CSV URPL do importu bazy leków

Import bazy leków:
- Panel ADMIN → Baza leków → Import CSV
albo
- jeśli drugs_catalog jest pusta i CSV leży w folderze projektu, aplikacja spróbuje auto-importu przy starcie.

Domyślne konto (tylko przy pustej bazie):
  admin / admin
